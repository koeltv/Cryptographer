Conversion d'un octet/caractère vers un tableau de bits La fonction ByteToBits() prend un octet/caractère et donne en sorti un tableau de bits correspondant à sa valeur en binaire.

Note
Le bit de poids le plus faible se trouve en position 0
Parameters
byte- caractère/octet à convertir
Returns
tableau de bits correspondant à la valeur en binaire de l'octet

Example

int byte[8] = ByteToBits('c');
for (int i=7; i>=0; i--) printf("%d", byte[i]); //On aura 01100011
